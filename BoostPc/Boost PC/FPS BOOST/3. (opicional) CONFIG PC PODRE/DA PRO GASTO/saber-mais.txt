configuração de usuário para o jogo Fortnite. Ele contém várias opções e configurações relacionadas ao jogo. Aqui está uma explicação geral do que algumas dessas opções podem significar:

- FullscreenMode: Define o modo de exibição em tela cheia (0 = desativado, 1 = ativado).
- CosmeticStreamingEnabled: Ativa ou desativa o streaming de cosméticos (0 = desativado, 1 = ativado).
- FortniteReleaseVersion: Versão de lançamento do Fortnite (provavelmente relacionada à versão do jogo).
- UnlockConsoleFPS: Define se o FPS (frames por segundo) está desbloqueado no console (False = desativado).
- SubGameSelectCount_Athena: Número de vezes que o modo de jogo "Athena" foi selecionado.
- SubGameLastSelectedTime_Athena: Data e hora da última seleção do modo de jogo "Athena".
- SubGameSelectCount_Campaign: Número de vezes que o modo de jogo "Campaign" foi selecionado.
- SubGameLastSelectedTime_Campaign: Data e hora da última seleção do modo de jogo "Campaign".
- CachedPlayerLevel: Nível do jogador em cache.
- bShowActorsWithSeasonItemTagMapIndicators: Define se os atores com indicadores de temporada são exibidos (True = ativado).
- CachedBattleStars: Número de estrelas de batalha em cache.
- CachedAlienStylePoints: Pontos de estilo alienígena em cache.
- CachedStylePoints: Pontos de estilo em cache.
- CachedClaimableRewards: Recompensas disponíveis em cache.
- CachedHighestBattlePassUnlockedPage: Página mais alta desbloqueada no passe de batalha.
- bShowCareerTabBang: Define se o indicador de notificação está visível na guia de carreira (False = desativado).
- bMotionBlur: Ativa ou desativa o desfoque de movimento (False = desativado).
- bShowGrass: Define se a grama é exibida (False = desativado).
- bShowFPS: Define se o contador de FPS é exibido (False = desativado).
- bUseGPUCrashDebugging: Ativa ou desativa a depuração de falhas da GPU (False = desativado).
- bStopRenderingInBackground: Define se a renderização é interrompida quando o jogo está em segundo plano (False = desativado).
- bRayTracing: Ativa ou desativa o Ray Tracing (False = desativado).
- b120FpsMode: Define se o modo de 120 FPS está ativado (False = desativado).
- FrontendFrameRateLimit: Limite de taxa de quadros (FPS) para a interface do usuário do jogo (120 neste caso).
- bIsEnergySavingEnabledIdle: Define se o modo de economia de energia está ativado quando o jogo está inativo (True = ativado).
- bIsEnergySavingEnabledFocusLoss: Define se o modo de economia de energia está ativado quando o jogo perde o foco (True = ativado).

Essas são apenas algumas das configurações presentes no código, e cada uma delas pode ter um efeito diferente no jogo, permitindo que os jogadores personalizem sua experiência de acordo com suas preferências.
